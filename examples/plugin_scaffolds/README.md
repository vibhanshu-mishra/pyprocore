# Plugin Scaffold Samples

These files show safe placeholder output for Phase 11D plugin developer scaffolding.

They are metadata examples only:

- No credentials or tokens
- No package installation
- No remote fetching
- No automatic module loading
- No generated-code execution
- No Procore API calls

Use `procore-sdk plugins scaffold dry-run --name example_local_plugin --output-dir ./tmp-plugin` to preview a scaffold before writing files.
