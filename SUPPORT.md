# Support

PyProcore is an independent open-source SDK. It is not official Procore support
and is not affiliated with or endorsed by Procore Technologies.

## Usage Questions

For setup, configuration, CLI usage, examples, or SDK questions, open a GitHub
discussion if enabled or a GitHub issue using the most relevant template.

Before opening a question, check:

- [README.md](README.md)
- [examples/README.md](examples/README.md)
- [docs/recipes/](docs/recipes/)
- [docs/automation/](docs/automation/)

## What to Include

Please include:

- PyProcore version
- Python version
- operating system
- command or code used
- expected behavior
- actual behavior
- sanitized logs with secrets removed
- whether a live Procore API call was involved

Do not paste access tokens, refresh tokens, client secrets, `.env` contents,
token stores, Authorization headers, or private project data.

## Out of Scope

This project cannot provide:

- official Procore account support
- Procore permission changes
- OAuth app approval or installation help inside your Procore account
- guarantees about undocumented Procore API behavior
- emergency production support

For Procore account, permission, billing, or official API support, contact
Procore through their official support channels.
